# TweenNode

Composable tween sequence node with editor preview and nested loop actions for Godot 4.

## Installation

1. Copy this `tween_node` folder into your project's `addons/` directory.
2. Enable **TweenNode** in **Project -> Project Settings -> Plugins**.
3. Add `TweenNode` to your scene and assign a `TweenSequence`.

## What's Included

- `plugin.cfg` / `plugin.gd`: editor plugin entry.
- `scripts/`: runtime and editor scripts.
- `tests/`: GdUnit4 tests used during development (excluded from release archives).
